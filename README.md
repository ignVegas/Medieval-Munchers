# Medieval-Munchers
2D Java Game for Advanced Software Design

The GamePanel class is a crucial component in a 2D game, serving as the main panel for rendering graphics and managing game elements. It includes a game loop implemented in the run method, responsible for updating and rendering game components at a specified frame rate. The game features a player (Player class) controlled by user input, enemies (Skeleton class), and various objects (mainObject class). The panel handles collision detection, player health, extra lives, and win/lose conditions. The code initializes the game world with tiles and objects, and the tick method manages the game's core logic, such as updating player and enemy positions. The paintComponent method is responsible for rendering the game world, objects, and entities on the panel. In summary, the GamePanel class encapsulates the fundamental functionality of a 2D game, including player input, object interactions, and graphical rendering.
